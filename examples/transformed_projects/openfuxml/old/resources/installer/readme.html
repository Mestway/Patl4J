Systemvorraussetzungen

openFuXML benötigt einige frei verfügbare Programme, die für einen reibungslosen Ablauf auf dem Computer eingerichtet sein müssen. Neben dem Java Development Kit ab Version 5 sind auch das Satzsystem LaTex (inkl. tex4ht), Ghostscript und Imagemagick Vorraussetzung für den Betrieb von openFuXML. Es folgen nun konkrete Informationen für die verschiedenen unterstützten Betriebssysteme.

Windows

Die hier vorgeschlagene Konfiguration funktioniert direkt mit dem Standard openFuXML. Sollten Sie andere Pfade für die Installation benutzen, müssen Sie die Datei tex4ht_win.env im Verzeichnis applications/fuxml/formats/math/htlatex entsprechend anpassen.

tex4ht

Bitte laden Sie sich hier das ZIP Archiv herunter und entpacken Sie es in das Verzeichnis C:\tex4ht

LaTex

Bitte laden Sie sich hier die MikTeX Installationsdatei herunter und installieren Sie das Basissystem nach C:\Programme\MikTex. Achten Sie bei der Installation unbedingt darauf, dass fehlende Pakete ohne zu fragen nachgeladen werden.

GhostScript

Bitte laden Sie sich hier die GhostScript Installationsdatei herunter und installieren Sie damit GhostScript.

ImageMagick

Bitte laden Sie sich hier unter Binaries die ImageMagick Windows Installationsdatei (...-Q16-windows-dll.exe Variante) herunter und installieren Sie damit ImageMagick. Achtung Windows installiert standardmässig ein Tool zum konvertieren von FAT32 Partitionen in NTFS Partitionen mit. Die Datei heisst convert.exe, genau wie die von openFuXML benötigte Datei aus dem ImageMagick Paket. Bitte nennen Sie die Windows-eigene Datei (in einer Standardinstallation zu finden unter C:\WINDOWS\system32\convert.exe) um, z.B. in convertfs.exe.

Anpassen der PATH Variablen

Damit openFuXML die benötigten Programme auch findet, müssen diese in der PATH Umgebungsvariablen stehen. Dazu gehen Sie bitte in der Systemsteuerung auf den Punkt System, dort im Reiter Erweitert auf Umgebungsvariablen und erweitern Sie die PATH Variable um den Ordner MikTex/miktex/bin, den bin Ordner Ihrer GhostScript Installation und den Pfad zu ImageMagick.

Linux (Debian/Ubuntu)

Die Installation der Programme unter Linux gestaltet sich dank des aptitude Paketmanager der Debian-artigen Distributionen sehr einfach. Bitte installieren Sie mit apt-get install die folgenden Pakete:
  • sun-java5-jdk
  • tetex-base
  • tetex-extra
  • latex-ucs
  • tex4ht
  • gs
  • imagemagick
